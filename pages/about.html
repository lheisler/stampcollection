About this site

The intention of this work was to do image searches on a set of stamps, in bulk, and capture the results in an organized pages. This should allow the user to easily review the results and get information about each individual stamp based on the reverse image search. Results are based on the image, and won't necessarily give information on the stamp. Hit results are often e-bay, or stamp collector sites, which may provide information on the potential value of these stamps.

Images of stamps were uploaded to the free image server (https://imgbb.com/), where they could be accessed by the serper API (https://serpapi.com/). A python script (search.py), sent each image link to the serper API, to do a reverse image search. Search results for each stamp were returned in JSON format, and contained up to 60 hits, providing information on the source of the hit and a path to the thumbnail image. A simple website was then generated with the make_htmls.py python script, producing one page for each stamp and a table of the results, with links ot the source pages. An index page with images of all stamps was generated, linking to each of the stamp pages. The content of the this site was given minimal review or curation, and is offered as-is, but open to feedback. Image search results may not be exact, with hits representing similar images, but hopefully the user will be presented with some that are correct.

This was a small project to understand better how to use reverse image-search APIs, and organize the results in a meaningful way. I hope the content of this site is interesting to the user, and provides some information about the variety of stamps that had been collected.
